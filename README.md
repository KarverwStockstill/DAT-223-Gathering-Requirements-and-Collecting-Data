# DAT-223-Gathering-Requirements-and-Collecting-Data
Gathering Requirements and Collecting Data

In this course, I learned how to bridge the gap between raw data and business solutions by identifying data needs, developing client proposals, and preparing datasets for Python / SQL analysis. I practiced requirements gathering by determining the organizational problems to solve and identifying the key data points needed for analysis. I also learned how to format data and create codebooks by processing synthetic datasets, such as banking fraud data, so variables could be imported and analyzed effectively. In addition, I applied basic programming tools, including Python with Pandas and SQL, to extract, organize, and present data in readable DataFrames. I also developed skills in forming research questions, proposing data collection methods, preparing status reports, cleaning and categorizing datasets, identifying indicators of suspicious activity, and presenting findings clearly.
